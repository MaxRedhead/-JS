Описание примера: При введении запроса и клике на «Искать», кнопка button type="submit" отправляет данные с формы обработчику формы. Как уже говорилось ранее, обработчиком формы обычно является страница сервера со скриптом для обработки введенных данных (в нашем случае — Яндекс). Обработчик формы указывается в атрибуте action. Атрибут target отвечает за то, откроется результат отправленной формы в новой вкладке или текущем окне. Чтобы результат формы открылся в новой вкладке, используйте значение _blank.

ВАЖНО! Каждый элемент формы должен иметь атрибут name для отправки.
